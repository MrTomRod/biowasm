biowasm

A repository of C/C++ genomics tools, pre-compiled to WebAssembly for use in a browser

Usage
<script src="https://cdn.biowasm.com/aioli/latest/aioli.js"></script>
<script>
let samtools = new Aioli("samtools/1.10");

document.write("Loading...");
samtools
    // Initialize samtools
    .init()
    // Run "samtools view" command with "-q 20" filter
    .then(() => samtools.exec("view -q 20 /samtools/examples/toy.sam"))
    // Output result
    .then(d => document.write(`<pre>${d.stdout}</pre>`));
</script>
Tools available

Supported tools include samtools, bedtools, seqtk, fastp, wgsim and more.
